# ARIS Stage0 Demo And Visual Handoff

This branch is a sanitized handoff package for the current ARIS Stage0 and dual-object visual progress.

Contents:

- `scripts/stage0_web_research.py`: Stage0 research prior CLI.
- `pipeline/research_prior_utils.py`: helper for consuming research priors.
- `docs/`: Stage0 workflow and handoff documentation.
- `runtime_prompts/`: reproduction prompt templates and guide with placeholders only.
- `outputs/aris_stage0_and_visual_results_20260515/`: compact demo package and selected visual results.

The package intentionally excludes local runtime folders, server credentials, API keys, model caches, and large raw outputs.
